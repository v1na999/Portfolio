MINI TRAFFIC LIGHT AND SMART MOVABLE BRIDGE SIMULATION

Project created in Arduino. The general objective of the project is to control the traffic light System and automated movable bridge. It represents a three-way cross road traffic and traffic light signal. It uses Light Dependent Resistor. It created by automated movable bridge with stepper motor triggered by ultrasonic sensor.
